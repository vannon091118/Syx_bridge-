Die Versions-Highlights sind ein Button im Header-Bereich mit goldenem Hintergrund via CSS-Klasse .version-highlight-btn die background-color: #d4a017 setzt. Der Hover-Effekt nutzt CSS-transition für einen Farbwechsel zu einem helleren Goldton, verstärkte box-shadow, und transform: scale(1.02). Der grüne Pulse-Dot daneben ist ein CSS-keyframe das mit animation: pulse 2s infinite läuft — opacity wechselt zwischen 0.4 und 1.0, transform scale zwischen 0.8 und 1.2. Das Modal nutzt Fade-In via CSS transition auf opacity von 0 auf 1 über 200ms. Der ESC-Listener in JavaScript prüft event.key === Escape und ruft closeVersionHighlights auf. Das Click-outside-Schliessen vergleicht event.target mit modal — wenn der Klick auf den Hintergrund trifft, wird geschlossen. Alles in Vanilla-JavaScript ohne Libraries — DOM-Manipulation via getElementById und style.display, keine Events via jQuery sondern native addEventListener. Das fresh-readme.js Script nutzt einen Regex ^v?\d|\(?\d+\.\d+ der Zeilenanfänge auf Versionen prüft, mit Pre-Filter der BU-, B4-, DOKU-, COMMIT-Tags ausschliesst. Der .gitignore wurde um core/.commit_msg.txt erweitert. Gemini hatte einen Social-Sharing-Tracker im GUI-Code — ein img-Tag mit tracking-Parameter. Entfernt.
