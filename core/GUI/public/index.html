v0.22.0 Released 2026-06-22 · P0/P1/P2 Härtung
📌 Highlights dieser Version
  • 🏷️FeatureLanguage-Tag — Mod-Name erhält Sprach-Suffix (z.B. "Orini Race DEUTSCH"). Info-Feld: Translation Credit automatisch gesetzt.
  • 🚨Fix P0__OVERWRITE Critical — V71-Dateien zerstörten Vanilla-DE-Texte. Behoben: Plugin gibt leeren Header zurück.
  • ⚡Fix P0Basis-Fallback — Bei komplettem Provider-Ausfall (NVIDIA 429, FCM offline, Groq Müll) wird jetzt die beste vorhandene DB-Übersetzung genutzt statt Englisch zu exportieren.
  • 🗑️Fix P1Groq Garbage-Detection — [1,2,3,...]-Batches als Fehler erkannt. Provider nach 2 Müll-Batches aus Routing ausgeschlossen.
  • 🛡️Fix P1Shield-Token-Preservation — __SHLD_N__ Tokens werden jetzt IMMER als erste Critical Rule in Prompts eingetragen. 233 Leaks eliminiert.
  • 📡Fix P2Path-Validierung modsOverride — ungültige Pfade via existsSync gefiltert statt Laufzeitfehler zu werfen.
  • 🎯FeatureisFreeModel() Provider-bewusst — OpenRouter, NVIDIA, Groq, Gemini werden korrekt als Free/Paid klassifiziert. Kein Name-Heuristik mehr.
  • 🧹Infra5 Thin-Wrapper entfernt (callGroqBatch etc.), Auto-Modus friert PRIMARY_MODEL nicht mehr ein, DB-Timeout auf 15s erhöht.
  • 📊FeaturerankModel() DB-gestützt — Modell-Ranking basiert jetzt auf echten avg_quality-Metriken statt String-Heuristik.
  • 📄DokuSSOT-Verletzungen behoben, CHANGELOG gesplittet, Commit-Layer vollständig überarbeitet (verify_commit_msg, update_plot, get_sidejoke).
